# 📊 Pandas Data Analysis Project

## 📌 Overview
This project demonstrates data analysis and manipulation using **Pandas** with the California Housing dataset.

It covers:
- Data loading
- DataFrame creation
- Data exploration
- Statistical analysis
- Feature engineering basics
- Data export

---

## 📂 Dataset
The project uses the built-in **California Housing dataset** from Scikit-learn.

---

## ⚙️ Steps Performed

### 1. Load Dataset
- Imported dataset using `fetch_california_housing`

### 2. Create DataFrame
- Converted dataset into Pandas DataFrame

### 3. Data Exploration
- head(), tail()
- shape, info()

### 4. Data Cleaning Checks
- Missing values check using isnull()

### 5. Statistical Analysis
- mean, std, min, max
- describe()

### 6. Feature Engineering
- Added new column (Price)

### 7. Data Selection
- iloc indexing

### 8. Correlation Analysis
- Relationship between features

### 9. Export Data
- Saved dataset as CSV and Excel
